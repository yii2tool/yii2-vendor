Git
===

Можно массово отправлять или получать изменения пакетов.

Для оптравки:

```
cd vendor/yii2tool/yii2-vendor/bin
php bin git/push
```

Для получения:

```
cd vendor/yii2tool/yii2-vendor/bin
php bin git/pull
```

Настройка приватного ключа на Windows разбирается в [этой статье](https://habrahabr.ru/post/266667/).
